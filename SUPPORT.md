# Support

Before opening an issue, please check the resources below.

---

## Documentation

Each repository has its own `README.md` with setup and usage instructions. Check there first.

---

## Getting Help

| Channel | Use for |
|---------|---------|
| [GitHub Discussions](https://github.com/orgs/ptheuslab/discussions) | Questions, ideas, general help |
| [Bug Reports](https://github.com/ptheuslab/.github/blob/main/ISSUE_TEMPLATE/bug_report.md) | Reproducible bugs |
| [Feature Requests](https://github.com/ptheuslab/.github/blob/main/ISSUE_TEMPLATE/feature_request.md) | Suggestions and improvements |
| security@ptheuslab.dev | Security vulnerabilities only |

---

## Before Asking

1. Search existing [issues](https://github.com/search?q=org%3Aptheuslab&type=issues) and [discussions](https://github.com/orgs/ptheuslab/discussions)
2. Check if the problem exists on the latest version
3. Include relevant details: environment, steps to reproduce, logs

---

*For security issues, see [SECURITY.md](SECURITY.md) — do not post them publicly.*
